# 📊 Smart Attendance System

A web-based Smart Attendance System that helps manage and track student attendance efficiently using modern web technologies.

---

## 🚀 Features

- ✅ Mark attendance (Present/Absent)
- 📅 View attendance records
- 📊 Dashboard for attendance tracking
- 🔍 Search student records
- 🧾 Clean and simple UI

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

---

## 📂 Project Structure

```
/frontend
/backend
/models
/routes
```

---

## ⚙️ How to Run the Project

### 1. Clone the repository
```bash
git clone https://github.com/Dnyaneshsasale/smart-attendance-system.git
```

### 2. Install dependencies
```bash
npm install
```

### 3. Start the server
```bash
npm start
```

### 4. Open in browser
```
http://localhost:3000
```
